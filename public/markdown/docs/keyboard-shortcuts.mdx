## Keyboard Shortcut Reference

Across Osopcloud, there's an assortment of keyboard shortcuts that you can use to navigate, set settings, and more.

---

### Across the App

- Go Home
  - `control + /`
  - `command + /` (Mac)
- Go to Settings
  - `control + shift + ,`
  - `command + shift + ,` (Mac)
- Print with Dynamic Printing (if enabled)
  - `control + P`
  - `command + P` (Mac)
