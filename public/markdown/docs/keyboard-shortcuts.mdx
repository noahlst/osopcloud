## Keyboard Shortcut Reference

Across Osopcloud, there's an assortment of keyboard shortcuts that you can use to navigate, set settings, and more.

---

### Across the App

- Go Home
  - `control + /`
  - `command + /` (Mac)
- Go to Settings
  - `control + shift + ,`
  - `command + shift + ,` (Mac)
- Show the Back Button on Large Windows
  - `control + alt + ←`
  - `command + option + ←` (Mac)

---

### Operating System Pages

- Go to the Project Website
  - `control + shift + p`
  - `command + shift + p` (Mac)
- Go to the Source Repository
  - `control + shift + r`
  - `command + shift + r` (Mac)
