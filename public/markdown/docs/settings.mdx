## Settings

With Settings, you can customise and configure Osopcloud.

Settings are stored on your device and remain the same every time you use Osopcloud.

To go to Settings, select the gear icon button in the right of the navigation bar.

There's a variety of settings available that change how Osopcloud looks. Some change looks, for example the session theme toggle. Others change how it works, for example, Focus Mode minimises notifications.

---

### Reset Osopcloud

To restore the default settings, select "About the Osopcloud Application" and then the "Reset Osopcloud" button.
